前言

Mexion（以下简称"我们"、"Mexion" 或"平台"）尊重并致力于保护用户的个人信息与隐私。本《隐私政策》旨在向您清晰、透明地说明我们在您使用 Mexion 服务（以下简称"服务"）过程中，如何收集、使用、存储、共享、跨境传输和保护与您相关的信息，以及您依法享有的相关权利。

请您在使用服务前仔细阅读本政策，特别是以加粗形式标注的条款，这些内容涉及您个人信息处理的重要事项。

您注册账户、点击同意按钮、调用任何 API 或以其他方式使用服务，即视为您已充分阅读、理解并同意本政策全部内容。若您不同意本政策任何内容，请立即停止使用服务。

本政策与《用户服务协议》共同构成您与我们之间的完整协议，相关术语定义以《用户服务协议》为准。

一、关于本政策的适用范围

  • 本政策适用于您通过任何方式（网站、API、SDK、控制台、移动应用、第三方集成等）访问或使用 Mexion 服务的全部场景。
  • 本政策不适用于：
  • 您所对接、调用的第三方 AI 模型提供商（如 OpenAI、Anthropic、Google、DeepSeek 等）及其各自的数据处理行为，该等行为受其各自隐私政策约束；
  • 您嵌入或链接到的第三方网站、产品与服务；
  • 其他用户对您信息的处理行为。
  • 我们鼓励您同时阅读您所使用的第三方服务的隐私政策，以充分了解您的信息在整个服务链路中的处理情况。

二、我们收集的信息

为向您提供稳定、安全、合规、可持续优化的服务，我们将基于"必要、合理、最小化"原则收集以下类别的信息：

2.1 您主动提供的信息

  • 账户信息：电子邮箱、用户名、密码（经加密存储，我们不可逆地知晓您的明文密码）、组织名称、联系人信息、所在地区等；
  • 身份验证信息：必要时为满足合规要求（如反洗钱、防欺诈）所需的身份证明、企业资质、税务信息等；
  • 支付与计费信息：账单地址、发票抬头、税号、银行账户或支付账户的部分信息（完整支付信息由第三方支付机构处理，我们仅获取交易结果与必要识别信息）；
  • 客户支持信息：您通过邮件、工单、聊天等渠道与我们沟通时提供的内容；
  • 您主动提交的其他信息：例如订阅推送、问卷调研、活动报名等。

2.2 您在使用服务过程中产生的信息

  • 服务调用数据：API 请求与响应、调用时间、调用模型、Token 用量、错误码、调用来源、调用频率、并发数等；
  • 请求内容数据：您通过 API 提交的提示词（Prompt）、参数、附加文件，以及对应的模型返回结果。

    • 关于该等数据的处理方式、保留期限与控制权，请重点参见本政策第四条与第七条。

  • 使用偏好与配置：您在控制台设置的偏好、自定义配置、密钥管理、用量提醒规则等；
  • 行为日志：您在控制台的访问路径、点击行为、停留时间、操作记录等。

2.3 我们自动收集的技术信息

  • 设备与环境信息：IP 地址、设备类型、操作系统、浏览器类型与版本、屏幕分辨率、语言设置、时区等；
  • 网络信息：访问来源、ISP、连接类型、网络状态、CDN 节点等；
  • 诊断信息：崩溃报告、性能数据、错误堆栈、调试信息等；
  • Cookies 与类似技术：详见本政策第十一条。

2.4 我们从第三方获取的信息

在符合法律的前提下，我们可能从下列来源获取与您相关的信息：

  • 支付服务提供商提供的交易状态信息；
  • 反欺诈、风险评估、合规审查服务商提供的风险标识；
  • 您授权链接的第三方账户（如使用第三方账号登录时）；
  • 我们的关联方与合作伙伴依法共享的信息；
  • 公开数据源（如企业信用信息系统）。

三、我们如何使用您的信息

我们将基于下列目的处理您的信息。通过使用服务，您理解并同意我们在下列范围内处理您的信息：

3.1 服务提供与基础运营（必要处理）

  • 创建、维护、管理您的账户；
  • 接收、路由、转发您的 API 请求至相应的第三方模型；
  • 返回响应结果、生成调用日志、统计用量；
  • 计算费用、生成账单、处理支付与对账；
  • 发送服务相关通知、安全提醒、账单提醒；
  • 提供客户支持与售后服务。

3.2 安全保障与风险控制（必要处理）

  • 验证身份、防范账户被盗用；
  • 检测异常调用、滥用行为、攻击行为；
  • 防范欺诈、洗钱、规避制裁等违法行为；
  • 满足审计、合规与监管要求；
  • 调查违反本政策或《用户服务协议》的行为；
  • 保护我们、关联方、其他用户和公众的合法权益。

3.3 服务改进与质量优化

  • 分析服务使用情况、性能指标、用户体验数据；
  • 测试新功能、修复缺陷、优化系统稳定性；
  • 进行用户调研、数据洞察、产品迭代；
  • 持续改进路由策略、调用效率、成本结构。

3.4 研究开发与产品演进

  • 基于服务运行数据进行内部研究、技术分析、模型评测、特征工程、模式识别、统计建模等技术工作；
  • 开发新功能、新产品、新模型路由策略与相关衍生服务；
  • 评估和优化我们对接的各模型供应商之服务质量；
  • 在合法合理范围内形成衍生数据、统计成果或技术资产用于产品演进及商业发展。

3.5 沟通与运营推广

  • 在您同意或法律允许的范围内，向您发送产品更新、活动信息、营销内容；
  • 您可随时通过控制台或邮件方式拒绝此类推送（不影响必要的服务类通知）。

3.6 法律合规与权利保护

  • 履行法律法规规定的义务；
  • 配合司法机关、监管机构的合法要求；
  • 在争议、诉讼、仲裁中主张或抗辩权利。

3.7 处理目的的合理扩展

若我们需要将信息用于本政策未明确列示但与原始收集目的具有合理关联的其他用途，我们将在符合适用法律的前提下进行处理；若该等用途超出原始目的的合理范围，我们将另行征得您的同意。

四、关于服务调用数据的特别说明

  • 您通过 API 调用本服务所传输的 Prompt、参数与模型返回结果（以下简称"调用数据"），其处理遵循下列原则：
  • 传输与执行：我们必然需要接收、传输调用数据，以将其转发至您所选择的第三方模型供应商并将结果返回给您。该等处理是服务的核心，您注册即视为同意；
  • 日志与运营：我们会留存调用元数据（如时间、模型、Token 数、错误码等）与必要的内容样本，用于计费核对、故障排查、滥用检测、合规审计与服务质量保障；
  • 改进与研究：在符合适用法律的前提下，我们可对调用数据进行内部分析、统计建模、特征提取等技术处理，以改进路由策略、提升服务质量、防范风险、研发新功能。该等处理通常以去标识化、聚合化或样本化形式进行；
  • 第三方约束：调用数据在传输至第三方模型供应商时，将同时受该等供应商隐私政策与数据政策的约束，我们对该等第三方的处理行为不承担责任。
  • 企业用户与高敏感场景：若您有更严格的数据处理需求（例如医疗、金融、政府场景），请与我们另行签署数据处理协议（DPA）或企业服务协议。未签署专项协议的，默认适用本政策的标准规则。
  • 避免提交敏感信息的责任：您应自行评估并避免通过本服务提交不应被存储或处理的高敏感信息（如未脱敏的身份证号、银行卡号、医疗记录、商业机密等）。因您主动提交此类信息所产生的任何风险与后果，由您自行承担。

五、关于去标识化、匿名化与衍生数据

  • 去标识化数据：指经过技术处理后，在不借助额外信息的情况下无法识别到特定个人的数据。
  • 匿名化数据：指经过技术处理后无法识别特定个人且不能复原的数据。
  • 聚合数据：指由多用户数据汇总形成、不指向任何特定个人的统计性数据。
  • 重要授权：经匿名化、聚合化处理后的数据不再属于您的个人信息。您理解并同意：我们对该等去标识化、匿名化或聚合数据，享有完整的使用、复制、分析、修改、再许可、转让权利，可用于任何合法目的（包括但不限于内部研究、产品开发、商业分析、对外披露行业报告、形成数据资产等），且无需另行通知或征得您同意。
  • 衍生数据：基于您的信息所形成的统计结果、分析报告、特征模型、技术参数等衍生成果（不指向特定个人的部分），其知识产权归我们所有。

六、信息共享、披露与转让

我们不会向第三方出售您的个人信息。但在下列情形下，我们可能依法共享、披露或转让您的信息：

6.1 基于业务必要的共享

  • 第三方模型供应商：在您发起 API 调用时，相关请求内容必然需要传输给您所选择的模型供应商，以完成模型推理与结果返回；
  • 服务提供商：与协助我们运营服务的技术合作方共享必要信息，包括但不限于：
  • 云计算与基础设施供应商；
  • 支付与结算服务商；
  • 客户支持与工单系统；
  • 数据分析、监控与可观测性平台；
  • 反欺诈、安全与合规服务商；
  • 邮件、短信、通知推送服务商；
  • 律师、会计师、审计师等专业服务机构。

我们将通过协议或其他方式，要求上述合作方对所获信息承担保密义务并仅在授权范围内使用。

6.2 关联方共享

我们可与关联方共享您的信息，以提供整合服务、统一管理、协同运营、风险控制及实现本政策所述其他目的。关联方对该等信息的处理应遵循不低于本政策的标准。

6.3 法律要求

在下列情形下，我们可在不另行通知您的情况下披露相关信息：

  • 适用法律法规明确要求；
  • 司法机关、行政机关、监管机构的合法调查、调取、协助要求；
  • 为执行本政策、《用户服务协议》或保护我们、关联方、其他用户的合法权益；
  • 为防范、调查欺诈、安全事件、违法行为或紧急情况；
  • 在国际制裁、出口管制合规审查中所需披露的范围。

6.4 业务变更与转让

在我们发生合并、分立、收购、重组、控制权变更、资产或业务出售（无论部分或全部）、破产清算等情形时，您的信息可作为相关交易资产的一部分被转让至受让方或继受人。我们将在适用法律要求范围内，就该等转让作出相应通知（若适用），但通知不构成您同意继续接受本政策的前提。受让方或继受人应继续遵守本政策（或不低于本政策保护标准）的约束。

6.5 经您同意的其他共享

在征得您单独同意或您明确指示的情况下，我们可向特定第三方共享信息。

七、信息存储、保留期限与跨境传输

7.1 存储位置

为支持服务的全球化运行与高可用性，我们可能将您的信息存储于我们或合作伙伴位于中国大陆、中国香港、新加坡、美国、欧盟或其他国家/地区的服务器中，并在不同司法管辖区之间进行传输与处理。

7.2 跨境传输

您注册并使用本服务，即视为您明确同意我们及合作方就本政策所述目的进行的跨境数据传输。我们将依据适用法律采取合理保护措施（如签署标准合同条款、采取技术加密、合作方资质审查等）。

7.3 保留期限

我们将根据下列原则确定信息保留期限：

| 信息类别 | 默认保留期限 |

|---|---|

| 账户信息 | 您账户存续期间及账户注销后合理期限内 |

| 调用元数据（用量、计费） | 自产生之日起 不少于 3 年（用于计费、审计、合规） |

| 调用内容样本 | 根据服务类型与合规需求，最短 30 天，最长可延至 3 年 |

| 财务、税务、合规记录 | 按适用法律要求（通常不少于 5 年或 10 年） |

| 安全与风控日志 | 不少于 6 个月 |

| 法律争议相关数据 | 至争议解决并超过法定时效 |

在下列情形下，我们有权超出上述期限继续保留相关信息：

  • 适用法律法规要求或允许更长保留期；
  • 处理待决或可能发生的争议、诉讼、调查、监管要求；
  • 为保护我们、关联方或其他用户的合法权益；
  • 已完成去标识化/匿名化的数据（参见第五条），不受本款保留期限限制。

7.4 账户注销后的处理

您注销账户后，我们将在合理期限内删除或匿名化您的可识别个人信息。但下列信息我们有权继续保留：

  • 涉及计费、税务、合规、审计的记录；
  • 用于法律争议、潜在调查的相关数据；
  • 已完成去标识化/匿名化的数据（不再视为个人信息）；
  • 系统备份中按正常备份循环周期存留的数据。

八、信息安全保障

  • 我们采取符合行业通行水平的技术与组织措施保护您的信息，包括但不限于：
  • 传输加密（TLS/SSL）；
  • 存储加密与密钥分级管理；
  • 访问控制、最小授权原则、多因素认证；
  • 网络隔离、入侵检测、日志审计；
  • 定期安全评估、漏洞扫描与渗透测试；
  • 员工保密协议、安全培训、权限审计。
  • 请您理解：
  • 互联网环境本身不可能保证绝对安全，我们无法对信息安全作出绝对保证；
  • 您应妥善保管您的账户凭证、API 密钥，因您自身原因造成的安全事件由您承担相应责任；
  • 若发生影响您合法权益的安全事件，我们将依法及时启动应急响应，并在适用法律要求范围内进行通知。

九、您的权利

在适用法律允许的范围内，您就您的个人信息享有下列权利：

9.1 基本权利

  • 查询与访问权：您可查询、获取我们持有的关于您的个人信息；
  • 更正与补充权：您可要求更正不准确或不完整的信息；
  • 删除权：在特定情形下，您可要求删除您的信息；
  • 限制处理权：在特定情形下，您可要求限制处理；
  • 撤回同意权：对基于同意的处理，您可撤回同意（但撤回不影响撤回前已进行的处理，亦不影响基于其他合法基础的处理）；
  • 可携权：以结构化、通用格式接收您的部分信息；
  • 拒绝自动化决策权：您可拒绝完全依赖自动化决策对您产生重大影响的处理；
  • 投诉权：您有权向有权监管机构投诉。

9.2 行使方式

请通过 support@mexion.local 联系我们行使上述权利。我们将在收到请求后于合理期限内回复（一般为 15-30 个工作日，复杂情况可适当延长并向您说明）。

9.3 身份验证

为保护您的信息安全，在响应您的请求前，我们将要求您提供必要的身份验证信息。无法完成验证的请求，我们有权拒绝。

9.4 合理限制

在下列情形下，我们有权拒绝、限制或部分响应您的请求：

  • 与履行法定义务相关；
  • 与国家安全、国防安全直接相关；
  • 与公共安全、公共卫生、重大公共利益相关；
  • 与刑事侦查、起诉、审判和执行判决等相关；
  • 涉及商业秘密或第三方合法权益；
  • 请求明显不合理、重复或滥用权利；
  • 履行请求将对其他个人或组织的合法权益造成重大损害；
  • 已完成去标识化/匿名化处理的数据，不属于本条权利的覆盖范围。

十、未成年人保护

  • 我们的服务仅面向年满 18 周岁或所在司法管辖区成年年龄的用户。
  • 我们不会有意收集未成年人的个人信息。若我们发现已无意中收集了未成年人信息，将及时删除。
  • 若您是未成年人的监护人，发现未成年人未经您同意使用了我们的服务，请通过 support@mexion.local 联系我们处理。

十一、Cookies 与类似技术

  • 我们使用 Cookies、Web Beacon、LocalStorage、SDK 等技术，用于：
  • 维持您的登录状态与会话；
  • 记住您的偏好设置；
  • 分析网站性能与使用情况；
  • 改进用户体验；
